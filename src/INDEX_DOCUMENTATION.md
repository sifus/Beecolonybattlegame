4. Consulter `/REGLES_DU_JEU.md` si besoin

### 🧪 JE VEUX TESTER LA V3
1. Lire `/GUIDE_TEST_V3.md`
2. Suivre la checklist (30-45 min)
3. Noter les bugs
4. Remplir le template de rapport

### 🔧 JE VEUX COMPRENDRE LA V3
1. Lire `/SESSION_COMPLETE_25_OCT.md` (vue d'ensemble)
2. Lire `/CHANGELOG_V3_COMPLET.md` (détails)
3. Approfondir avec :
   - `/REFONTE_VISUELLE_V3.md`
   - `/RETOUR_GENERATION_ALEATOIRE.md`

1. Commencer par `/README_V3.md`
2. Lire `/SESSION_COMPLETE_25_OCT.md`
3. Lire `/CHANGELOG_V3_COMPLET.md`
4. Parcourir toute la doc par ordre chronologique :
   - Session 1 : Gameplay (voir `/CHANGELOG_SESSION.md`)
   - Session 2 : UX (voir `/MODIFICATIONS_UI.md`)
   - Session 3 : Menu (voir `/NOUVELLES_FONCTIONNALITES.md`)
   - Session 4 : V3 (tous les fichiers ci-dessus)

### 🔙 JE VEUX RESTAURER UNE VERSION
1. Choisir version :
   - V1 : Voir `/backup-v1-stable/restore.md`

2. Suivre instructions de restauration
