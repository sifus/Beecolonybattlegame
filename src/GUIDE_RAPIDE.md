# 🎮 GUIDE RAPIDE - Rush

## 🚀 Démarrage Rapide (30 secondes)

### Votre Objectif
Détruire **toutes les ruches ennemies** (rouges) pour gagner ! 🏆

### Comment Jouer

#### 1️⃣ Sélectionner des Abeilles
- 🖱️ **Clic sur votre arbre** (gauche) → Sélectionne toutes vos abeilles
- 🔵 **Clic-glisser** (cercle jaune) → Sélection multiple libre

#### 2️⃣ Envoyer des Abeilles
- 🖱️ **Clic simple** sur un arbre → Envoie les abeilles sélectionnées
  - Sur arbre **neutre** (gris) → Les abeilles gravitent autour
  - Sur arbre **ennemi** (rouge) → Combat automatique ⚔️

#### 3️⃣ Construire des Ruches
- 🖱️🖱️ **Double-clic** sur un arbre avec vos abeilles autour
  - **5 abeilles** = 1 ruche niveau 1 (7 HP)
  - **20 abeilles** = Amélioration niveau 2 (35 HP)

---

## ⚠️ DANGER : Étangs

### 💧 Zones Bleues = Mort !
- **1 abeille sur 3** (33%) tombe dans l'eau
- **Contournez les étangs** pour éviter les pertes
- Ou envoyez **+50% d'abeilles** pour compenser

---

## 🎯 Stratégies de Base

### ✅ À FAIRE
- ✅ Construire des ruches rapidement (plus de production)
- ✅ Protéger vos abeilles (ne pas les laisser seules)
- ✅ Contourner les étangs
- ✅ Attaquer les ruches ennemies faibles

### ❌ À ÉVITER
- ❌ Laisser des abeilles idle dans un coin (l'IA les attaque !)
- ❌ Traverser les étangs sans raison
- ❌ Attaquer trop tôt (attendre d'avoir assez d'abeilles)
- ❌ Oublier de réparer vos ruches endommagées

## 📊 Informations Utiles

### Production d'Abeilles
- **Ruche niveau 1** : 1 abeille toutes les 3 secondes
- **Ruche niveau 2** : 1 abeille par seconde
- ⚠️ Pas de production si combat en cours

### Coûts
| Action | Coût |
|--------|------|
| Construire ruche niveau 1 | 5 abeilles |
| Améliorer L1 → L2 | 20 abeilles |
| Réparer 1 HP | 1 abeille |
| Détruire ruche L1 | 7 abeilles |
| Détruire ruche L2 | 35 abeilles |

### Santé des Ruches
- **Niveau 1** : 7 HP maximum
- **Niveau 2** : 35 HP maximum
- Les ruches endommagées continuent de produire

## 🤖 Comprendre l'IA

L'ennemi (abeilles rouges) a une stratégie :

1. **Priorité #1** : Terminer ses constructions en cours
2. **Priorité #2** : Réparer ses ruches endommagées
3. **Priorité #3** : Capturer des arbres neutres
4. **Priorité #4** : Attaquer vos abeilles isolées
5. **Priorité #5** : Attaquer vos arbres

**💡 Conseil** : Attaquez pendant que l'IA construit !

## 🏁 Déroulement d'une Partie Typique

### ⏱️ 0-30s : Début de Partie
- Vous commencez avec **0 abeilles** (production démarre)
- Attendez d'avoir **~10 abeilles**
- Capturez un **arbre neutre** proche

### ⏱️ 30s-2min : Expansion
- Construisez votre **première ruche** (5 abeilles)
- Capturez **2-3 arbres neutres**
- Construisez des ruches sur chaque arbre
- **Améliorez** quelques ruches au niveau 2 (20 abeilles)

### ⏱️ 2-5min : Combat
- Accumulez **30-40 abeilles** sur vos ruches L2
- Identifiez une **ruche ennemie faible**
- Envoyez vos abeilles (contournez les étangs !)
- **Détruisez** la ruche (7 HP pour L1, 35 HP pour L2)
- Capturez l'arbre devenu neutre

### ⏱️ 5min+ : Domination
- Répétez jusqu'à détruire **toutes les ruches ennemies**
- Protégez vos territoires
- **Victoire !** 🎉

## 💡 Astuces Avancées

### 🧠 Tactiques
- **Rush Early** : Impossible avec 0 abeilles au début (équilibré)
- **Économie** : Plus de ruches L2 = plus d'abeilles = victoire
- **Map Control** : Capturer les arbres centraux donne l'avantage
- **Pond Baiting** : Forcer l'ennemi à traverser les étangs

### ⚔️ Combat
- Envoyez **10-20% de plus** que nécessaire (marge de sécurité)
- Attaquez en **vagues** si l'ennemi a beaucoup d'abeilles
- Détruisez les ruches **une par une** (focus fire)
- Réparez vos ruches **immédiatement** après combat

### 🗺️ Positionnement
- Gardez vos abeilles **autour de vos arbres**
- Ne laissez **jamais** d'abeilles idle seules
- Contrôlez les **arbres sans étangs** pour expansion facile

## 🎓 FAQ

**Q : Comment je gagne ?**
A : Détruire toutes les ruches ennemies (HP → 0).

**Q : Les abeilles attaquent-elles automatiquement ?**
A : Oui, sur les arbres ennemis. Non sur les neutres/alliés.

**Q : Puis-je annuler une construction ?**
A : Non, les abeilles sont consommées immédiatement.

**Q : Comment réparer une ruche ?**
A : Double-clic sur l'arbre avec des abeilles autour. 1 abeille = +1 HP.

**Q : Pourquoi je ne peux pas améliorer mon arbre de départ ?**
A : Les arbres de départ (gauche/droite) ne peuvent PAS être améliorés.

**Q : Les étangs affectent-ils l'IA ?**
A : Oui ! L'IA perd aussi 33% de ses abeilles dans les étangs.

**Q : Combien d'abeilles envoyer pour détruire une ruche L2 ?**
A : 35 abeilles minimum. Mais envoyez 50+ si vous traversez un étang (33% de pertes).
