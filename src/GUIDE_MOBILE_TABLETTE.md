# Guide d'Utilisation Mobile & Tablette

## 🎮 Rush - Jeu de Stratégie

---

## 📱 Installation et Lancement

### Sur Mobile (iPhone/Android)

1. **Ouvrez le jeu dans votre navigateur**
   - Safari (iOS) ou Chrome (Android)

2. **Activez le mode paysage**
   - Tournez votre téléphone horizontalement
   - Si un message apparaît, suivez les instructions

3. **Passez en plein écran**
   - Dans le menu principal, appuyez sur **"Plein Écran"**
   - Cela cache les barres du navigateur pour une meilleure expérience

4. **Commencez à jouer !**
   - Appuyez sur **"Jouer"**

---

## 🎯 Contrôles Tactiles

### Sélectionner des Abeilles/Lucioles

#### Méthode 1: Tap sur un arbre allié
- **Tap simple** sur un arbre avec vos abeilles
- Toutes les abeilles de cet arbre sont sélectionnées
- Un cercle orange apparaît autour d'elles

#### Méthode 2: Tap-Glisser (cercle de sélection)
- **Tap et maintenez** avec votre doigt
- **Glissez** pour créer un cercle
- Toutes les abeilles dans le cercle sont sélectionnées
- Relâchez pour confirmer

### Envoyer des Abeilles

- **Tap simple** sur un arbre de destination
- Les abeilles sélectionnées s'y rendent automatiquement

### Construire/Réparer une Ruche

- **Double-tap rapide** sur un arbre allié ou neutre
- Vos abeilles autour de l'arbre construisent/réparent
- **Coût:** 5 abeilles pour une ruche niveau 1
- **Amélioration:** 20 abeilles pour passer au niveau 2

## ⚠️ Conseils Importants

### Mode Paysage Obligatoire
- Le jeu est conçu pour être joué **horizontalement**
- En mode portrait, un message vous demandera de tourner l'appareil
- La surface de jeu s'adapte automatiquement à votre écran

### Plein Écran Recommandé
- Pour une meilleure immersion
- Plus d'espace visible
- Pas de barre d'adresse qui gêne

### Zoom Désactivé
- Le zoom par pincement est désactivé
- Le double-tap ne zoome pas (il construit)
- Cela évite les actions involontaires

### Pas de Sélection de Texte
- Vous ne pouvez pas sélectionner le texte par erreur
- Fini le surlignage bleu accidentel !
- Cela rend le jeu plus fluide

## 🎯 Astuces Mobile

### Pour Mieux Jouer

1. **Utilisez vos deux pouces**
   - Maintenez l'appareil à deux mains
   - Pouce gauche pour sélectionner
   - Pouce droit pour donner les ordres

2. **Tapez précisément**
   - Les zones tactiles sont optimisées
   - Pas besoin d'appuyer fort

3. **Le double-tap demande de la pratique**
   - Tapez rapidement deux fois
   - Trop lent = deux taps simples
   - Trop rapide = un seul tap

4. **Anticipez les mouvements**
   - Les abeilles mettent du temps à se déplacer
   - Planifiez à l'avance

5. **Attention aux étangs**
   - 33% de vos abeilles se noient
   - Contournez-les !

## 🐛 Problèmes Connus

### Le plein écran ne fonctionne pas (iOS Safari)
**Solution:** 
- C'est normal, iOS nécessite un geste utilisateur
- Appuyez sur le bouton "Plein Écran" dans le menu
- Ou ajoutez à l'écran d'accueil pour mode PWA

### Le jeu est trop petit/grand
**Solution:**
- Le jeu s'adapte automatiquement
- Rafraîchissez la page
- Tournez l'appareil en paysage

### Le double-tap ne fonctionne pas
**Solution:**
- Tapez plus rapidement
- Assurez-vous d'avoir des abeilles autour de l'arbre
- Vérifiez que l'arbre est allié ou neutre

### Message "Tournez en paysage" ne disparaît pas
**Solution:**
- Tournez complètement votre appareil
- Attendez 1-2 secondes
- Verrouillez la rotation si nécessaire

## 💡 FAQ Mobile

### Puis-je jouer en portrait ?
❌ Non recommandé. Le jeu est optimisé pour le paysage.

### Puis-je installer le jeu ?
⚠️ Pas encore, mais les fondations PWA sont là.

### Le jeu consomme-t-il beaucoup de batterie ?
⚡ Non, consommation modérée grâce aux optimisations.

### Puis-je jouer hors ligne ?
❌ Pas pour le moment, connexion nécessaire pour les sons.

### Y a-t-il des contrôles tactiles spéciaux ?
✅ Non, les contrôles standards suffisent (tap, double-tap, glisser).

### Le jeu fonctionne-t-il sur tous les mobiles ?
✅ Oui, navigateurs modernes (2023+) requis.
